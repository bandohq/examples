# Bando API Examples

## Installation

### Node

```bash
brew install node
```

### Install PNPM

```
brew install pnpm
```


```bash
pnpm install
```

## Usage


### Configuration

Create a file named `.env` with the following content:

```env
EVM_RPC_URL=ff
REFERENCE=dd
EVM_PRIVATE_KEY=
``` 

```bash
pnpm run evm
```

```bash
pnpm run solana
```
