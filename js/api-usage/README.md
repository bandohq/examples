# Bando API Examples

## Installation

### Node

```bash
brew install node
```

### Install PNPM

```
brew install pnpm
```


```bash
pnpm install
```

## Usage


### Configuration

Create a file named `.env` with the following content:

```env
EVM_RPC_URL=ff
REFERENCE=dd
SOLANA_PRIVATE_KEY=
``` 

# EVM networks

```bash
pnpm run evm
```


# SVM networks
```bash
pnpm run solana
```
